# CitiesTranslatePinyin  
将文本内的的中文城市名称以首字母大写排序，并额外生成关联外键，转化为Json格式。  
也可以用于排序人名、商标等信息。

```angular2html
@Json预览  
    # all_citys_info = [  
    #     { A: {  
    #         city: [],  
    #         foreign: int,  
    #     }},  
    #     { B: {  
    #         city: [],  
    #         foreign: int,  
    #     }},  
    #      ...  
    # ]  
```
